# 金融数据分析与建模 · 小组作业提交仓库

> 课程主页：<https://lianxhcn.github.io/>  
> 课程仓库：<https://github.com/lianxhcn/dsfin>  
> 学生提交指南：[STUDENT_GUIDE.md](./STUDENT_GUIDE.md)

本仓库收集和展示各届同学提交的小组作业，按学年归档，每个小组拥有独立文件夹。

## 目录结构

```
dsfin-hw/
├── submissions/
│   ├── 2026/
│   │   ├── T01-G01-股票收益预测/
│   │   └── T01-G02-信用风险建模/
│   └── 2027/
│       └── ...
├── README.md
├── STUDENT_GUIDE.md
└── TEACHER_GUIDE.md
```

文件夹命名：`T{话题编号}-G{小组编号}-{简短主题}`，例如 `T01-G03-资产定价`。
